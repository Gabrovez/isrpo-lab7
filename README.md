# Лабораторная работа №7: Адаптивная верстка с использованием Flexbox
### Описание
Лабораторная работа посвящена освоению современных технологий CSS-разметки — модуля Flexbox для создания адаптивных веб-интерфейсов. Адаптивная верстка позволяет веб-страницам автоматически подстраиваться под размеры экрана различных устройств (компьютер, планшет, смартфон).
### Структура проекта
Проект содержит следующие файлы и директории:
> Lab7_Adaptive_FIO/
>>index.html - Базовая HTML-разметка для изучения свойств Flexbox

### Основные концепции Flexbox
#### Flex-контейнер и элементы
Flexbox — одномерная система компоновки для размещения элементов в строку или столбец.
```css
.container {
  display: flex;
}
```
#### Свойство `flex-direction`
Определяет направление главной оси контейнера.
```css
.container {
  flex-direction: row;           /* элементы в строку → (по умолчанию) */
  flex-direction: row-reverse;   /* элементы в строку ← */
  flex-direction: column;        /* элементы в столбец ↓ */
  flex-direction: column-reverse;/* элементы в столбец ↑ */
}
```
#### Свойство `justify-content`
Выравнивание элементов по главной оси.
```css
.container {
  justify-content: flex-start;   /* элементы в начале (по умолчанию) */
  justify-content: flex-end;     /* элементы в конце */
  justify-content: center;       /* элементы по центру */
  justify-content: space-between;/* равномерно, крайние у краёв */
  justify-content: space-around; /* равномерно, отступы по краям */
  justify-content: space-evenly; /* равномерно, одинаковые отступы */
}
```
#### Свойство `align-items`
Выравнивание элементов по поперечной оси.
```css
.container {
  align-items: stretch;    /* растягивает элементы по высоте (по умолчанию) */
  align-items: flex-start; /* выравнивание в начале */
  align-items: flex-end;   /* выравнивание в конце */
  align-items: center;     /* выравнивание по центру */
  align-items: baseline;   /* выравнивание по базовой линии текста */
}
```
#### Свойство `flex-wrap`
Управление переносом элементов при нехватке места.
```css
.container {
  flex-wrap: nowrap;      /* без переноса (по умолчанию) */
  flex-wrap: wrap;        /* перенос на новую строку */
  flex-wrap: wrap-reverse;/* перенос в обратном направлении */
}
```
#### Свойство `gap`
Современный способ задания отступов между элементами.
```css
.container {
  gap: 10px;      /* одинаковые отступы по горизонтали и вертикали */
  gap: 10px 20px; /* вертикальные 10px, горизонтальные 20px */
}
```
#### Свойства элементов
`flex-grow` - определяет, как элемент растягивается для заполнения свободного пространства.
```css
.item {
  flex-grow: 1; /* элемент займёт всё доступное пространство */
}
```
`flex-basis`- задаёт базовый размер элемента перед распределением свободного пространства.
```css
.item {
  flex-basis: 200px; /* начальная ширина элемента */
} 
```
`flex-shrink`- определяет, как элемент сжимается при нехватке места.
```css
.item {
  flex-shrink: 1; /* элемент может сжиматься (по умолчанию) */
}
```
Сокращённая запись `flex`
```css
.item {
  flex: 1 1 250px; /* grow | shrink | basis */
}
```
`align-self` - Индивидуальное выравнивание отдельного элемента по поперечной оси.
```css
.item-special {
  align-self: flex-end; /* выравнивание только этого элемента в конце */
}
```
### Практические примеры
#### Навигационное меню
Горизонтальное меню с равномерным распределением ссылок.
```html
<nav class="navbar">
  <a href="#">Главная</a>
  <a href="#">Услуги</a>
  <a href="#">Цены</a>
  <a href="#">Контакты</a>
</nav>
```
```css
.navbar {
  display: flex;
  justify-content: space-between;
}
```
#### Карточки товаров 
Адаптивная сетка карточек с автоматическим переносом и выравниванием контента.
```html
<div class="cards">
  <div class="card">
    <h3>Товар 1</h3>
    <p>Описание товара</p>
    <button>Купить</button>
  </div>
  <!-- ещё карточки -->
</div>
```
```css
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}
.card {
  display: flex;
  flex-direction: column;
  flex: 1 1 250px;
}
.card button {
  margin-top: auto;
}
```
#### Центрирование элемента
Абсолютное центрирование любого элемента по горизонтали и вертикали.
```html
<div class="centered-content">
  <h2>Центрированный блок</h2>
  <p>Текст внутри центрированного контейнера</p>
</div>
```
```css
.centered-content {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  height: 300px;
}
```
